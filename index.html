Web scraping and Data
Analysis in Python

In this Project I webscraped over 1500 Data Analyst Jobs in USA posted on Indeed site using python packages Request and BeautifulSoup, performed data cleaning manipulation and then performed exploratory data analysis using the pandas library. I Finally made Visualizations using matplotlib and plotly trying to answer the Questions, What is the most sought vizualization Tool? Tableau or Power BI? And for the programming language, is it Python or R?. From my Analysis, Tableau and PowerBI are the leaders in the Job Market with the former coming in as the First. And not surprisingly, Python is the most used as it is the more preferred Language.
